# 🔐 Caesar Cipher — Защита пароля

**СРС 1: Защита системы и данных в мобильных приложениях**

---

## 📌 Описание задачи

Реализовать шифрование пароля **без использования готовых криптографических библиотек**.  
Алгоритм — классический **Шифр Цезаря (Caesar Cipher)**.

---

## ⚙️ Как работает алгоритм

Шифр Цезаря сдвигает каждый символ строки на фиксированное число позиций в таблице ASCII.

| Параметр | Значение |
|----------|----------|
| Ключ сдвига | `7` |
| Размер таблицы | `128` (ASCII) |
| Шифрование | `(ord(символ) + 7) % 128` |
| Дешифровка | `(ord(символа) - 7) % 128` |

---

## 🔄 Пример работы

| Этап | Значение |
|------|----------|
| **Исходный пароль** | `MyP@ssw0rd!` |
| **Ключ сдвига** | `7` |
| **Зашифрованный пароль** | `T}W'zz\_>(` |
| **Расшифрованный пароль** | `MyP@ssw0rd!` |
| **Проверка** | ✅ `True` |

**Как это происходит (на примере символа `M`):**
- `M` → ASCII код = `77`
- `77 + 7 = 84`
- `84 % 128 = 84`
- `chr(84)` = `T` ← зашифрованный символ

**Дешифровка:**
- `T` → ASCII код = `84`
- `84 - 7 = 77`
- `chr(77)` = `M` ← оригинальный символ

## 🚀 Как запустить

**Требования:** Python 3.6+


## ⚠️ Важное замечание

Шифр Цезаря используется **исключительно в учебных целях** для демонстрации принципа подстановочного шифрования.  
Для реальной защиты паролей в production-приложениях необходимо использовать:
- `bcrypt` / `scrypt` / `Argon2` (хэширование паролей с солью)
- `AES-256` (симметричное шифрование данных)
- `Android Keystore` / `iOS Keychain` (хранение ключей)
